InfoNode Tabbed Panel
=====================

InfoNode Tabbed Panel is developed by NNL Technology AB. Visit
<http://www.infonode.net> for more information and the latest version.

InfoNode Tabbed Panel is a Java Swing based graphical component that
provides a notebook like interface for selecting a component from a
list of components. InfoNode Look and Feel is also included in the 
jar file.


Requirements
------------

InfoNode Tabbed Panel requires J2SE 1.4.0 or later. The latest Java
version can be downloaded from 
<http://java.sun.com/javase/downloads/index.jsp>.


Examples
--------

You will find a source code example application called SimpleTabbedPanelExample 
in "examples/net/infonode/tabbedpanel/example" that shows how to create 
a tabbed panel, titled tabs, modify properties and use themes. It is possible 
to compile and run the example.


Documentation
-------------

Documentation is available in HTML format, docs/index.html.
A tutorial is also available at <http://www.infonode.net>.
