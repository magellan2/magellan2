## Nightly Build

Dies ist eine experimentelle Magellanversion, ein sogenannter "nightly build". Hier probieren
wir neue Funktionen aus und stellen sie experimentierfreudigen Benutzern zur Verfügung. Diese
Version ist jedoch nicht so gut getestet wie eine stabile Version. Im schlimmsten Fall können
Daten (Reporte, Befehle...) verloren gehen. Falls Sie lieber auf Nummer sicher gehen wollen 
und auf die neuesten Funktionen verzichten können, benutzen Sie lieber eine stabile Version.
Für eine Liste mit allen Änderungen siehe CHANGELOG.txt.

## English version

This is an experimental version of Magellan, a so called nightly build. Here we try out new 
functions and make them available to adventurous users. This version has, however, not been
tested by as many people as a stable version. In the worst case, you could lose data
(reports, orders...). If you prefer to play it safe and you do not need the newest features, 
you may prefer to download a stable version. For a complete list of changes see CHANGELOG.txt
