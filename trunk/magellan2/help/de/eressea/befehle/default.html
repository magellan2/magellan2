DEFAULT "befehl"

Dieser Befehl ermöglicht es, für eine Einheit einen Befehl für die nächste Runde festzulegen. Dieser Befehl wird in die Zugvorlage der nächsten Runde als Default-Befehl eingesetzt.

Achtung! Der normale Default-Befehl wird damit überschrieben! Somit kann man sogar den Befehl NACH als Default setzen.
