The actions for the "Map" menu entry.

This package contains all actions for the menu entry "Map". Magellan provides the following menu:

  • Map
    • Set origin
    • Retrieve islands
    • Save as image
    • Select all
    • Select none
    • Invert selection
    • Select islands
    • Fill selection
    • Expand selection
    • Open selection
    • Add selection
    • Save selection as
