Menü Desktop

Das Desktop-Menü hat folgende Punkte:
  • Layout
  • Tabs verbergen
Innerhalb des Desktopmenüs findet man Funktionen, um das Aussehen von Magellan an die eigenen Wünsche anzupassen. Seit Magellan 2.0 gibt es ein neues Layout Konzept, das auf sogenannten Docks basiert. Dabei sind einzelne Elemente des Fensters frei verschiebbar und sogar vom Hauptfenster Magellans lösbar, um sie zum Beispiel auf einem zweiten Bildschirm zu platzieren.

Unterhalb der oben genannten Menüpunkte sieht man alle gerade für Magellan bekannten Docks. Jedes dieser Docks stellt einen bestimmten Teil von Magellan dar. Das kann die Regionsübersicht, die Befehle oder die Karte sein. Jedes dieser Docks besitzt am oberen Rand rechts einige Funktionsbuttons, die wir hier im folgenden genauer erklären....
