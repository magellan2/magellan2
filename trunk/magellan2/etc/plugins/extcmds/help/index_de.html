Dies ist der Editorbereich der Extended Commands.

Dieser Text soll etwas dabei helfen, die komplexen Möglichkeiten dieses Plugins zu verstehen. Dieser Editor basiert auf BeanShell und ist damit Java ähnlich. Es gibt Klassen und Methoden. Die Variablen sind typsicher.

Du hast die Wahl zwischen diversen Typen.

  • Einheiten - jede einzelne Einheit im Spiel
  • Container - z.B. Regionen, Schiffe, Burgen
  • Bibliothek - eine Sammlung nützlicher Funktionen für Einheiten und Container

Die API für die einzelnen Klassen ist sehr umfangreich. Prinzipiell kann man auf alle Teile von Magellan und dem gerade geladenen Report zugreifen. Theoretisch ist es sogar möglich, eigene Fenster zu öffnen, darauf wird diese Dokumentation aber nicht weiter eingehen. Für tiefere Einblicke in die API empfehlen wir folgende Seiten:

  • Magellan Library
    magellan.library.*
    Enthält eine Vielzahl der zumeist benötigten Klassen und Beschreibungen für Magellan.
  • Magellan Rules
    magellan.library.rules.*
    Enthält zahlreiche Definitionsdateien für das Regelwerk.
  • Extended Commands Helper
    Helper API
    Zeigt Methoden der Helfer-Klasse, die während der Ausführung von Scripten immer zur Verfügung steht.
  • Magellan2 API (gesamt)
    http://magellan.log-out.net/javadoc
    Dokumentation aller Klassen und Möglichkeiten im Magellan Client. Speziell interessant sind die Klassen in magellan.library.
  • Java 5 API
    http://java.sun.com/j2se/1.5.0/docs/api
    Eine Dokumentation der aktuell verwendeten Java Umgebung. All diese Klassen stehen zur Verfügung.
  • BeanShell Syntax
    http://www.beanshell.org/manual/contents.html
    Eine komplette Dokumentation der Möglichkeiten von BeanShell.
