InfoNode Look and Feel
======================

InfoNode Look and Feel is developed by NNL Technology AB. Visit
<http://www.infonode.net> for more information and the latest version
of the library.

InfoNode Look and Feel is a Java Swing look and feel based on Metal.
It's slimmer and has other colors than the standard Metal look and feel.


Requirements
------------

InfoNode Look and Feel requires J2SE 1.4.0 or later. The latest Java
version can be downloaded from 
<http://java.sun.com/javase/downloads/index.jsp>.


Documentation
-------------

The Javadoc for InfoNode Look and Feel is included in the docs 
folder of this distribution. A tutorial is also available at 
<http://www.infonode.net>.
