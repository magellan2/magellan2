Kräuterkunde und Alchimie

Mit Hilfe von allerlei Zutaten (vor allem Kräutern) können Alchimisten mächtige Tränke brauen. Ein solcher Trank kann von jeder Einheit benutzt werden, um verschiedene Effekte zu erzielen.

Kräuterkunde und Alchimie sind teure Talente! Um sie lernen zu können, muß der Kräuterkundige bzw. der Alchimist genügend Silber mit sich tragen (zum Thema Lernkosten siehe das Kapitel über Talente).

Kräuter

In jeder Region kann eine bestimmte Kräutersorte wachsen. Welche Kräuter in welchen Terrains wachsen können, kann dieser Tabelle nachgelesen werden. Jede Runde wachsen einige wenige (wenige bedeutet in diesem Fall: WIRKLICH SEHR WENIGE) Kräuter nach.

Diese Kräuter können von Einheiten, die über das Talent Kräuterkunde verfügen, mit dem Befehl MACHE KRÄUTER gesammelt werden. Pro Person der suchenden Einheit kann ein Kraut gefunden werden. Anders als bei anderen MACHE-Befehlen findet jedoch nicht jede Einheit, die nach Kräutern sucht, auch wirklich etwas. Allerdings steigt die Wahrscheinlichkeit, ein Kraut zu finden, mit dem Talentwert der Einheit an.

Hat eine Einheit aus ausreichend hohes Talentwert Kräuterkunde, so wird sie erkennen können, welche Kräuter in einer Region wachsen, oder gar wieviele Kräuter es in der Region tatsächlich gibt. (Dies geschieht automatisch. Es ist hierzu kein Befehl notwenig.)

Tränke

Aus Kräutern kann mittels MACHE trank eine alchimistische Tinktur hergestellt werden. Einen Überblick darüber, welche Tränke es gibt und was sie bewirken liefert diese Tabelle. Jeder Trank hat jedoch ein spezielles Rezept. Dieses erhält man, sobald ein Alchimist dem entsprechenden Talentwert erreicht (siehe unten).

Um einen Trank herstellen zu können, muß das Alchimietalent des Mischers doppelt so hoch sein wie die Stufe des Trankes. Ein Alchimist kann jeden Monat pro Level, die er besser ist als das Doppelte der Trankstufe, einen weiteren Trank brauen. Pro Monat kann ein Alchimist aber nur eine Sorte Trank brauen.

Hydrius Hirseblatt ist ein Trankbrauer mit Alchimie 9. Er kann also Tränke der Stufen 1 bis 4 herstellen. Angenommen er beschäftigt sich diesen Monat mit einem Wissenstrank (Stufe 3). Das Doppelte der Trankstufe ist 6, also ist Hydrius 3 (=9-6) Talentstufen besser. Dies bedeutet, daß er in diesem Monat 4 (=3+1) Wissenstränke anfertigen kann - vorausgesetzt daß er die passenden Ingredenzien auftreiben kann. Alternativ kann Hydrius 2 Stufe 4 Tränke, 6 Stufe 2 Träke oder 8 Stufe 1 Tränke anfertigen.

Einen fertigen Trank kann jede Einheit benutzen. Hierzu muß sie einfach den Befehl BENUTZE trank [einheit] geben. In der Regel profitiert die Einheit selbst von der Wirkung des Trankes. In einigen Fällen (z.B. Pferdesegen) wirkt ein Trank jedoch auch auf die gesamte Region.

Gibt man einen zusätzlichen Einheitenparameter an, so wird der Trank der entsprechenden Einheit verabreicht. Hat die Einheit HELFE gesetzt oder gehört gar der selben Partei an, so wird sie den Trank benutzen. Andernfalls wird das Tarnungstalent der Gebereinheit mit der Wahrnehmung der Empfängereinheit verglichen. Ist die Tarnung höher, so wird der Trank benutzt. Sollte jedoch die Wahrnehmung höher sein, so wechselt der Trank nur den Besitzer, als ob er mittels GIB übergeben worden wäre.

Derweil alle Tränke manuell (d.h. mittels BENUTZE) verwendet werden können, gibt es einige Tränke, die auch automatisch benutzt werden. Dies geschieht jedoch nur, wenn die Einheit, die den Trank benutzt, diesen selbst besitzt, d.h. z.B. daß ein Heiltrank nicht an eine schwer verletzte Einheit abgegeben wird.

  • Heilsalbe, Heiltrank: wird automatisch von einer Einheit verwendet, wenn diese durch einen Kampf Schaden genommen hat. Beide Produkte werden nur dann verwendet, wenn die Einheit so schwer verletzt ist, daß die Anwendung sinnvoll ist (d.h. wenn ihr mehr als 20 bzw. 300 Trefferpunkte fehlen.
  • Hirtenblut: wird automatisch von einer Vampireinheit getrunken, wenn die Einheit hungert.
  • Berserkerswut, Drachenhaut: wird automatisch von einer Einheit vor einem Kampf zu sich genommen. Dies geschieht nur, wenn die Einheit noch nicht unter der Wirkung eines anderes Trankes steht.
  • Ewiges Leben: wird automatisch von einer Einheit während eines Kampfes verwendet, wenn sie einen tödlichen Treffer erleidet.

In der Regel wirken Tränke für 10 Personen. Einige Gebräue haben jedoch andere Wirkungen. Wenn ein Trank angewandt wird, geht die überschüssige Wirkung nicht verloren. Sie wird solange aufbewahrt, bis sie aufgebraucht wird oder ein anderer Trank benutzt wird.

Eine Vampireinheit, die aus 50 Personen besteht, kann mit einer Anwendung von Hirtenblut (200 Vampire benötigen keine Bauern) 4 Monate lang ohne Bauern auskommen.
Eine Einheit aus 3 Personen, die zu Beginn eines Kampfes Berserkerswut zu sich nimmt, wird für die nächsten drei Kämpfe +3 auf das Kampftalent erhalten. Im 4. Kampf wird nur noch eine der Personen den Bonus erhalten.
