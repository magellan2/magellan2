Desktopkonfiguration

Die Oberfläche von Magellan kann flexibel an die eigenen Bedürfnisse angepasst werden. Dazu unterstützt Magellan drei Modi zur Darstellung der Fensterkomponenten:

  • Splitpane-Modus
  • Fenster-Modus
  • Layout-Modus

Diese Modi werden durch die Datei magellan_desktop.ini, welche im Magellan-Verzeichnis angelegt wird, konfiguriert. Dieses Kapitel erklärt die Verwendung und Konfiguration der Desktop-Modi.
Hier kann man sich eine beispielhafte Version der Datei magellan-desktop.ini anschauen, die eine Reihe von Konfigurationen enthält, die verschiedene Nutzer von Magellan für sich erstellt haben. Man kann auch seine Version der magellan-desktop.ini durch diese Version ersetzen und sich dann die verschiedenen Konfigurationen direkt in Magellan anschauen.
